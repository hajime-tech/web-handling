# テンション変動の対策

張力（テンション）変動は、しわ・蛇行・塗工むら・印刷見当ずれ・巻き不良など、ほぼ全てのウェブハンドリングトラブルの引き金となる。
本ページでは張力変動の発生源を分類し、診断方法と対策を整理する。

## 1. 張力変動の分類

時間スケールと原因で大きく分けられる。

| 種類 | 周波数帯 | 主原因 |
|------|---------|--------|
| DC（オフセット） | — | 設定値ずれ、巻径推定誤差、センサ校正ずれ |
| 低周波変動 | 0.1〜1 Hz | 制御ループの不安定、ダンサ脈動 |
| 中周波変動 | 1〜10 Hz | ロール偏心、軸受不良、ベルト振動 |
| 高周波変動 | 10〜100 Hz | モータトルクリプル、機械共振 |
| 過渡変動 | パルス状 | スプライス、加減速、外乱 |

## 2. 主な発生源

### (a) 制御系起因

- PIDゲイン不適切（過小：応答遅、過大：発振）
- センサ・アクチュエータの応答遅れ
- ループ周期が遅すぎる
- 巻径推定誤差
- 慣性補償・摩擦補償不足
- アンチワインドアップ不備

### (b) 機械系起因

- ロール偏心
- 軸受摩耗・打痕
- ベルト・チェーン・カップリングのバックラッシ
- ギヤ歯合いムラ
- ロール撓み・たわみ振動
- フレーム共振

### (c) ウェブ起因

- 厚さ分布（ゲージプロファイル）
- 密度・物性の幅方向ムラ
- 巻出ロールの偏心・段ズレ
- スプライス（接紙部）の通過

### (d) 環境起因

- 温度変動 → ウェブの寸法変化
- 湿度変動 → 紙の伸縮
- 外部振動（隣接設備、空調）
- 床振動

## 3. 周波数別の診断

張力変動は FFT で周波数解析すると、原因を切り分けやすい。

| 観測される周波数 | 原因の典型 |
|-----------------|-----------|
| ロール回転周波数 $f_r = V/(\pi D)$ | ロール偏心、軸受不良 |
| ロール回転 × $n$（高調波） | 軸受打痕、歯車かみ合い |
| 巻出ロール周波数 | 巻出原反のバギング、偏心 |
| 商用電源周波数（50/60 Hz） | 電気ノイズ、ハム |
| 制御周期の逆数 | PID 発振 |
| 機械固有振動数 | フレーム共振 |

### 計算例

ライン速度 $V = 200\,\text{m/min}$、ロール径 $D = 150\,\text{mm}$ の場合：

$$
f_r = \frac{200/60}{\pi \times 0.15} = 7.07\,\text{Hz}
$$

→ 7 Hz 付近に張力ピークが見えれば、このロールが原因の候補。

## 4. 制御系チューニング

### PID パラメータの最適化

[制御機器の選定](../tension/devices.md) で示した手順に従う。要点：

1. 積分のみ（弱比例）で安定動作確認
2. 比例ゲインを上げ、発振直前で50%戻す
3. 微分は通常不要
4. 巻径補正FFを追加
5. 加減速時の慣性補償FFを追加

### スパン剛性とゲイン

スパン剛性 $K_s = E w h / L$ は張力ループのゲインに直接効く。

- 短スパン・高剛性 → 制御が容易、応答速い
- 長スパン・低剛性 → 制御困難、ダンサ向き

スパン剛性が低い系では、ダンサロール方式が圧倒的に有利。

### モデルベース制御

複雑なライン（多軸同期、相互干渉）では、各ゾーンの動的方程式を立てて状態空間モデル化し、

- LQG（線形二次ガウス）制御
- MPC（モデル予測制御）
- H∞ ロバスト制御

などを導入する例も増えている。

## 5. 機械系の対策

### ロール偏心

- 旋盤での仕上げ振れを 10 μm 以下に
- 動バランス取り（バランシングマシン）
- ロール軸の真直度確認

### 軸受

- 高精度軸受（精度等級 P5 以上）
- 適正予圧、潤滑管理
- 振動センサで監視

### 駆動系のバックラッシ

- 低バックラッシ減速機（遊星、ハーモニック）
- カップリングはディスク式またはダイレクトドライブ
- ベルト駆動はタイミングベルトで剛性確保

### 配管・配線

- 油圧配管の気泡 → サーボ応答悪化
- ケーブル劣化 → ノイズ

## 6. ウェブ起因対策

### 原反品質

- 厚さ分布（ゲージ）の管理：±2% 以内が標準
- バギング検査：受入検査でロール表面のしわ・たるみ確認
- 残留応力測定（光弾性、X線回折）

### スプライス対策

- 接紙部の段差を最小化（突合せ、ラップ）
- 接紙時はアキュムレータで張力変動を吸収
- 接紙通過時に制御ホールド

### 巻出原反の偏心

- 受入時に外観検査
- 必要なら芯出し再巻取り

## 7. 環境対策

### 温湿度

- 工場恒温恒湿化：23±2℃、50±5%RH が理想
- 局所空調（塗工部、印刷部）
- ウェブ保管環境とライン環境を一致

### 振動

- 装置を独立基礎に
- 隣接設備（プレス、コンプレッサ）との距離・防振
- ライン全長で振動測定

## 8. 過渡変動の対策

### 加減速

- なめらかな加減速プロファイル（S字加減速）
- 加速度上限の設計値遵守
- 巻取慣性が大きい場合は時定数を長く

### スプライス

- アキュムレータ（蛇腹式、フェスツーン式）
- 接紙時の張力一時ホールド
- スプライス検出センサ

### 断ウェブ復帰

- 自動糸通し（air doctor、ロープ通し）
- アキュムレータからの再立上げ手順

## 9. 診断・解析ツール

| ツール | 用途 |
|--------|------|
| 高速ロードセル | 1 kHz オーダーの張力波形 |
| FFT アナライザ | 周波数別の振動・張力解析 |
| 振動計（加速度センサ） | 機械振動の特定 |
| データロガー | 長時間トレンド |
| PLC ハイスピードロギング | 制御系内部信号 |
| 高速度カメラ | ウェブ挙動の可視化 |
| 赤外線サーモグラフィ | 温度ムラの可視化 |

## 10. 対策チェックリスト

問題発生時の確認手順：

- [ ] 張力波形を高速ロードセルで記録
- [ ] FFT 解析で支配周波数を特定
- [ ] その周波数とロール周速、固有振動数を照合
- [ ] 制御系の P/I/D 値とループ周期を確認
- [ ] 巻径推定値と実測値を比較
- [ ] アクチュエータの応答（指令 vs 実値）を確認
- [ ] ウェブ起因か機械起因かを切り分け
- [ ] 環境条件（温湿度・振動）の記録

## 理解度チェック

??? question "演習1: 周波数からロール特定"
    ライン速度 $V = 120\,m/min$ で運転中、張力FFTに 4.24 Hz のピークが見えた。直径 D 何 mm のロール由来と推定されるか。

    ??? success "解答"
        $V = 2\,m/s$
        ロール回転周波数 $f_r = V/(\pi D)$ より、$D = V/(\pi f_r) = 2/(\pi \times 4.24) \approx 0.150\,m = 150\,mm$
        該当する 150 mm 径のロール（偏心・軸受不良が疑わしい）を点検。

??? question "演習2: PID チューニング"
    張力制御で「振動的な変動」が出ている。比例ゲイン $K_p$ を下げると振動は収まるが、外乱に対する応答も鈍くなる。次の手は？

    ??? success "解答"
        (1) **積分時間 $T_i$ を見直す**（短すぎると振動、長すぎると応答遅）
        (2) **巻径補正フィードフォワード** を追加（巻径変化への外乱を予測補償）
        (3) **加減速時の慣性補償FF** を追加
        (4) センサノイズが主因なら、ローパスフィルタを入れる
        $K_p$ を一律下げるのではなく、振動の周波数特性を見て対処する。

??? question "演習3: スプライス通過対策"
    巻出ロールの接紙部（スプライス）通過時に張力が乱れ、後段に張力変動が伝播してしまう。
    効果的な対策を 3 つ挙げよ。

    ??? success "解答"
        (1) **アキュムレータ**（蛇腹式／フェスツーン式）を入れ、機械的に張力変動を吸収
        (2) **接紙部の段差を最小化**（突合せ、ラップ厚を薄く）
        (3) **接紙検出センサ＋制御一時ホールド**（通過中は張力制御をホールドし、追従させない）
