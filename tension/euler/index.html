    オイラーの摩擦式

    ウェブハンドリングで最重要の式の一つが オイラーのベルト摩擦式（Euler-Eytelwein formula）。 ウェブがロールに巻き付いて滑らずに保持される条件、あるいはロール前後で張力差を維持できる限界を与える。

    1. 式の形

    ロールに 巻き付き角 \(\theta\) [rad] で巻き付いたウェブが、ロール表面で完全滑り直前にある状態を考える。 高張力側を \(T_2\)、低張力側を \(T_1\)（\(T_2 > T_1\)）とすると、

    \[ \boxed{\,\frac{T_2}{T_1} \le e^{\mu \theta}\,} \]

    ここで \(\mu\) はウェブとロール間の摩擦係数、\(\theta\) は ラジアン で表した巻き付き角。

    等号成立は滑り限界。実用上は安全余裕として 1.5〜2 倍の係数を掛けて設計する。

    2. 導出（要点）

    巻き付き角 \(\theta\) の微小角 \(d\theta\) に着目し、半径 \(R\)、線張力 \(T\) を考える。

    • 半径方向の力の釣り合い：\(dN = T\,d\theta\)
    • 接線方向：\(dT = \mu\, dN = \mu T\, d\theta\)

    これを \(\theta = 0\)（張力 \(T_1\)）から \(\theta\)（張力 \(T_2\)）まで積分：

    \[ \int_{T_1}^{T_2} \frac{dT}{T} = \mu \int_0^{\theta} d\theta \]
    \[ \ln \frac{T_2}{T_1} = \mu \theta \;\;\Longrightarrow\;\; \frac{T_2}{T_1} = e^{\mu \theta} \]
    flowchart LR
    A[低張力 T1] -->|巻き付き角 θ| B[高張力 T2]

    3. 数値感覚

    巻き付き角 180°（= \(\pi\) rad）の Sラップを想定し、\(\mu\) を変えると：

    摩擦係数 μ \(e^{\mu \pi}\) 意味
    0.05 1.17 ほぼ滑る。低摩擦フィルム
    0.10 1.37 フィルム平均
    0.20 1.87 ゴムロール上のフィルム
    0.30 2.57 良好
    0.50 4.81 ニップに近い
    1.00 23.1 ほぼ滑らない

    このため ドライブロールには高摩擦ゴム被覆、巻き付き角を稼ぐためにSラップ・ダブルラップを採用する。

    4. ウェブ・ロール間の摩擦係数

    実測の代表値（理想条件、参考値）：

    ウェブ ロール表面 μ
    PET 鏡面クロム 0.15〜0.25
    PET 梨地クロム 0.20〜0.35
    PET NBR ゴム HS70 0.30〜0.50
    OPP クロム 0.20〜0.30
    アルミ箔 クロム 0.10〜0.25
    紙 鋳鉄 0.20〜0.40
    紙 ゴム 0.30〜0.60

    注意点：

    • 速度依存性あり：高速ほど 空気同伴 で μ が低下する（後述）。
    • 湿度・温度で大きく変化。
    • 油分・離型剤がフィルムに残ると一桁低下することも。

    5. 空気同伴（air entrainment）

    高速ウェブはロール表面との隙間にエアを引き込み、エア層に乗ってしまう。

    \[ h_\text{air} = c \cdot R \left( \frac{\eta V}{T/w} \right)^{2/3} \]

    これは弾性流体潤滑（EHL）解析からの近似で、\(\eta\) は空気粘度、\(V\) は搬送速度、\(R\) はロール半径、\(c \approx 0.643\)（Knox-Sweeney 1977）。

    空気層厚 \(h_\text{air}\) がロール・ウェブの表面粗さの合計より大きくなると、実効摩擦係数 \(\mu_\text{eff}\) が急減し、オイラー式の保持力が崩れる。 高速対策：

    • 溝付きロール（マクロ溝で空気排出）
    • 粗面化ロール（接触点を確保）
    • ニップロール（強制接触）
    • 真空サクションロール（ロールに穴・真空源）

    6. 滑りの判定と「巻き付け（wrap）張力」

    ロール前後で実際に発生する張力差 \(T_2 / T_1\) が \(e^{\mu \theta}\) を超えると 巨視的滑り が起こり、駆動ロールであれば搬送速度を失う。 これを避けるための実用設計：

    \[ \frac{T_\text{high}}{T_\text{low}} \le \frac{e^{\mu \theta}}{SF} \]

    ここで \(SF\) は安全率（典型 1.5〜2.0）。

    7. 応用：駆動ロールの設計

    ドライブロールがスリップせずにライン速度を決めるための条件：

    1. 巻き付き角 \(\theta\) をできるだけ大きく取る（Sラップ、ダブルラップ）。
    2. ロール表面の \(\mu\) を高くする（ゴム被覆、粗面、セラミックなど）。
    3. 必要ならニップロールでメカニカルに挟む。
    4. 入口・出口の張力比をオイラー式で照査する。

    8. 応用：アンチスリップ計算例

    例：PET 12 μm、ライン速度 100 m/min、巻き付き角 180°、\(\mu = 0.2\)、入口張力 100 N/m、出口張力？

    オイラー式より \(T_2/T_1 \le e^{0.2 \pi} = 1.87\)。 よって出口張力上限 \(T_2 \le 187\,\text{N/m}\)。安全率 1.5 を見込むと実用上限 \(\approx 125\,\text{N/m}\)。 これを超える張力差を作りたければ、巻き付き角を増やすか、ニップを併用する。
